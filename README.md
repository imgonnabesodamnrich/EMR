## 1. 研究背景

民用航空液压泵是控制飞机起降和飞行姿态的核心动力源。由于运行环境恶劣，液压泵不可避免地会产生磨损、腐蚀等故障。基于深度学习的数据驱动方法在故障诊断中表现出了良好的潜力，因为它们不需要人工手动提取复杂的物理特征。

然而，在实际工业场景中，深度学习面临一个基本矛盾：**数据不平衡**。
液压泵在绝大多数时间里都处于健康运行状态，因此收集到的“健康数据”极其庞大；而真正发生故障的时间很短，导致“故障数据”非常稀少。如果直接用这种极度不平衡的数据去训练神经网络，模型会倾向于把所有样本都预测为“健康”，从而失去诊断故障的能力。

## 2. 现有数据增强方法的局限性

为了解决数据不平衡，通常的方法是“数据增强”，即人为制造一些少数类（故障类）样本。

在图像处理领域，数据增强非常直观，比如把一张杯子的图片旋转、平移或翻转，它依然是一个杯子（类别不变性）。但在**一维振动信号**领域，直接应用简单的变换可能会改变信号的物理特性。例如，如果盲目改变振动信号的幅值或频率，生成的信号可能就不再属于原来的故障类别，这会给深度学习模型提供错误的引导（即标签错误）。

## 3. 核心方法：经验模式重构（EMR）

针对上述问题，论文提出了一种针对一维振动信号的数据增强方法——**经验模式重构（Empirical Mode Reconstruction, EMR）**。它的核心思想是：在生成新样本的同时，保留原始信号的内在物理成分，确保新样本的多样性，且不改变其原有的故障类别。

EMR的具体步骤分为以下几个阶段：

1. **信号分解（EMD）**：
   振动信号通常是多种机械部件振动的混合体。首先，使用经验模态分解（EMD）将一段原始的故障振动信号分解成多个基本的**固有模态函数（IMF）**。这就好比把一段复杂的和弦声音，拆解成几个单音符的轨迹。
2. **随机加权生成多样性**：
   在分解出的多个IMF中，随机选择一个，并给它乘上一个随机的权重（按比例放大或缩小）。这使得最终组合出来的新信号与原信号有所不同，从而增加了数据的多样性，防止神经网络“死记硬背”导致过拟合。
3. **信号重构**：
   将改变权重后的IMF与其他未改变的IMF重新相加，组合成一条新的一维振动信号。
4. **标准差（STD）对齐（关键步骤）**：
   在机械故障诊断中，振动信号的“标准差（STD）”是判断设备是否发生故障以及故障严重程度的核心统计指标。为了保证生成的新信号没有脱离原有的故障类别，作者强制要求重构后的新信号经过数学缩放，使其标准差与原始采集的故障信号**完全一致**。

## 4. 结合深度学习的诊断流程

有了EMR方法后，整体的故障诊断框架如下：

1. 取出训练集中的少数类（故障样本），使用EMR进行批量扩充，直到故障样本的数量与健康样本的数量一致，形成一个类别平衡的训练集。
2. 将平衡后的数据集输入到卷积神经网络（ConvNet）中进行特征学习和分类。
3. 为了进一步提升模型泛化能力，论文在模型训练的每一个Epoch（轮次）中，都会对故障样本重新进行一次EMR随机增强。这意味着网络每次看到的故障数据细节都有微小的差异。

## 5. 实验结果

论文采用了真实的航空液压泵地面试验数据来验证方法。实验设置了1种健康状态和5种故障状态。在训练集中，健康样本有400个，而每种故障样本仅有4个（不平衡比例高达 100:1）。

实验对比了直接使用卷积网络（ConvNet）、向下采样（减少健康数据）、向上采样（直接复制故障数据）以及本文提出的 **EMR + ConvNet** 方案。

结果表明：
* 直接复制数据（向上采样）容易导致特征空间重叠，分类边界不清晰。
* **EMR + ConvNet** 的平均 F1 分数达到了 **97.01%**，相较于不进行数据增强的基线模型提高了约 4.39%。
* 通过高维特征可视化（t-SNE）可以直观观察到，使用EMR增强后，深度学习模型能够将不同类型的故障样本清晰地分隔在不同的区域，几乎没有类别重叠现象。

## 6. 总结

论文针对工业场景中常见的故障数据稀缺问题，提供了一个无需复杂先验知识的信号增强方案。经验模式重构（EMR）通过“分解-加权-重构-方差对齐”的机制，既保证了生成样本的多样性，又严格守住了样本的物理类别属性。该思路不仅适用于航空液压泵，对其他旋转及往复式机械设备的振动信号分析也具有直接的参考价值。
